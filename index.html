Network Classification by Geographical Scope

A comprehensive overview of computer network types based on their coverage area, from personal devices to global infrastructure

PAN

Personal Area Network

Range
1-10 meters
Speed
Up to 2 Mbps
Technology
Bluetooth, Infrared, USB, Zigbee
Use Cases
Wireless headphones, smart watches, file transfer between personal devices
Characteristics
Low power consumption, simple setup, device-to-device communication, temporary connections

LAN

Local Area Network

Range
Up to 1-2 kilometers
Speed
10 Mbps - 100 Gbps
Technology
Ethernet, Wi-Fi, Token Ring, FDDI
Use Cases
Office networks, school campuses, smart homes, library networks
Characteristics
High data transfer rates, low latency, private ownership, wired/wireless options

MAN

Metropolitan Area Network

Range
5-50 kilometers
Speed
10 Mbps - 10 Gbps
Technology
Fiber optics, Microwave, DSL, ATM
Use Cases
City-wide WiFi, cable TV networks, university campuses, government networks
Characteristics
Connects multiple LANs, public/private ownership, moderate cost, ISP managed

WAN

Wide Area Network

Range
Global (unlimited)
Speed
1 Mbps - 100 Gbps
Technology
Satellites, Fiber optics, Leased lines, MPLS
Use Cases
The Internet, multinational corporations, global banking, cloud services
Characteristics
Largest geographical area, highest infrastructure cost, multiple admin control, uses routers & switches

Key Insights & Trends

  • Network boundaries are blurring with technologies like 5G and IoT creating hybrid network architectures
  • Software-Defined WAN (SD-WAN) is revolutionizing enterprise network management and cost efficiency
  • Mesh network technology is extending traditional network topologies for better coverage and reliability
  • Edge computing is creating new network architecture paradigms by processing data closer to the source
  • Network virtualization allows multiple virtual networks to operate over shared physical infrastructure
